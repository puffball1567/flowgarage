# Intellectual Property Notes

FlowGarage v0.1.0 uses public, general software ideas: artifact metadata, report
sections, format rendering, validation, and callback-based publishing.

The implementation is intentionally small and original. It does not copy DSLs,
report templates, storage adapters, or application GUI code from other projects.

If a concern is raised about a specific feature or name, the project should
review it in good faith and remove or adjust the feature when appropriate.
