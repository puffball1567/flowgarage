# v0.1.0 Scope

FlowGarage v0.1.0 is releasable when the core can be used without any external
service.

Included:

- `GarageBundle`
- `ReportSection`
- `Artifact`
- validation
- source-record conversion helpers
- Markdown rendering
- JSON rendering
- HTML rendering with escaping
- package manifest generation
- `GarageOutcome`
- plugin manifests
- publisher callback registry
- tests, example, benchmark smoke test

Deferred:

- file publisher
- database publisher
- signed bundles
- application GUI surfaces
- FlowCaptain integration package
